== Description ==

**WP Google Dorks AI** est un plugin WordPress puissant qui permet aux utilisateurs de générer des requêtes Google Dork adaptées en fonction de leur contenu. Ce plugin utilise l'API OpenAI pour fournir des suggestions précises et optimisées, facilitant ainsi le référencement et la recherche.

**Fonctionnalités principales :**
- **Génération Automatique de Dorks :** Créez facilement des requêtes Google Dork contextuelles basées sur le contenu de vos articles.
- **Intégration API OpenAI :** Demandez des suggestions adaptées avec une clé API OpenAI.
- **Affichage Structuré :** Les résultats sont présentés de manière organisée, en classant les dorks par catégorie.
- **Compatible avec divers types de contenu :** Fonctionne avec les articles, pages et autres types de contenu WordPress.

== Installation ==

1. Téléchargez le fichier ZIP du plugin :
   Rendez-vous sur la page du plugin à l'adresse <https://kevin-benabdelhak.fr/plugins/wp-google-dorks-ai/> et téléchargez le fichier ZIP.

2. Uploader le plugin sur votre site WordPress :
   - Allez dans « Extensions » > « Ajouter »
   - Cliquez sur « Téléverser une extension »
   - Sélectionnez le fichier ZIP téléchargé puis cliquez sur « Installer maintenant »
   - Activez le plugin en cliquant sur « Activer ».

== Utilisation ==

1. Rendez-vous sur la page des options du plugin pour entrer votre clé API OpenAI.
2. Accédez à un article ou une page sur votre site.
3. Utilisez le générateur de Google Dorks pour obtenir des suggestions adaptées à votre contenu.
4. Les dorks générés seront affichés et organisés par catégorie.

== FAQ ==

= Que sont les Google Dorks ? =
Les Google Dorks sont des requêtes de recherche avancées conçues pour extraire des résultats spécifiques et pertinents de Google, souvent utilisées dans le cadre de l'optimisation pour les moteurs de recherche (SEO).

= Est-ce que cela fonctionne avec tout type de contenu ? =
Oui, le plugin est compatible avec tous les types de contenu, y compris les articles de blog, les pages, etc.

= Une clé API est-elle requise ? =
Oui, vous aurez besoin d'une clé API OpenAI pour utiliser les fonctionnalités de génération de dorks.
